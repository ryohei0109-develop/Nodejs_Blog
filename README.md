# Node.js ブログアプリケーション

## ファイル構造

- app.js (各種制御処理)
- public (CSS、画像等を配置)
    - css
      - style.css
    - images
- views (画面ファイル(.ejs)を配置)
    - article.ejs (記事詳細)
    - header.ejs (共通ヘッダー)
    - list.ejs (記事一覧)
    - login.ejs (ログイン画面)
    - top.ejs (トップページ)

## 事前準備
npmインストール  
$npm install ejs  
$npm install express  
$npm install express-session  
$npm install mysql
